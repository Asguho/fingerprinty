# Fingerprinty
A basic fingerprinting library.
## Example
### Browser
```js
import { getId } from 'https://esm.sh/jsr/asguho/fingerprinty';
console.log(await getId());
```
### Node
```bash
npx jsr add asguho/fingerprinty
```
```js
import { getId } from 'fingerprinty';
console.log(await getId());
```
